###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: NaN 

<i>In the sum function, it attempts to add the values of a and b. However, when you call sum(1, "2"), JavaScript tries to add a number (1) to a string ("2"), which is not a valid operation. This results in a "NaN" value (Not-a-Number) because JavaScript cannot perform arithmetic operations between different data types like numbers and strings.</i>

</p>
</details>

###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: ['🍕', '🍫', '🥑', '🍔'] ?

<i>info.favoriteFood initially points to the first element of the food array, which is "🍕." However, when you later assign "🍝" to info.favoriteFood, it does not affect the food array. Therefore, the food array remains unchanged, and the output is still ['🍕', '🍫', '🥑', '🍔'].</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer:B: Hi there, undefined 

<i>because the sayHi function expects a name parameter to be passed when calling it. Since no argument is provided in sayHi(), the name inside the function remains undefined, resulting in the output "Hi there, undefined."</i>

</p>
</details>

```javascript
});

console.log(count);
```

- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: 2

</p>
